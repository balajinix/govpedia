govpedia.in

Hello there! This project is an attempt to make publicly available government data accessible!

This website has 3 parts.

1. Search - govpedia.in/search - still under construction.

This is an installation of Elasticsearch. I considered using Apache Solr and Hibernate Search. I chose ElasticSearch because its got tailwind, is used by Wikimedia for its search feature, and seems to have a lot better API.

2. Wiki - govpedia.in/wiki - still under construction.

This is an installation of the MediaWiki software.

3. Share - govpedia.in/share

This is an installation of Pligg. Its works similar to digg.com. I had used this software for rightnews.in and smartvote.in/share, both of which no longer exist.
