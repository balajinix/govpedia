Using IOT and Big Data Analytics for Smart Governance


1. Chattisgarh Govt has been using RFIDs for ensuring supply of PDS ration.

2. The E-way bill system in GST, which has been piloted in Karnataka, involves using RFID based monitoring of vehicles, so that they don't evade tax. GST Council has said that this will be implemented across India in 2018. This presents opportunity for automatically detecting anomalies using Big Data.

3. Improving the efficiency of Road transport using Analytics - Traffic police do use CCTVs, speed guns, and Character recognition (of number plates) for catching violations. And to identify traffic blockades. By crowd sourcing Dash Cam feeds, with help from Uber/Ola, and logistic companies, traffic violations can be reported at scale. Helmet less driving, rash driving etc in city roads. Right lane hogging on Highways, which reduces the speed of all vehicles.
